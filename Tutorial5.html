DEFINING PREFORMATED TEXT

Sometime using break tag is not convenient.Alternatively, you can use the Preformated tag tag to display spaces, tabs, line breaks, etc. exactly as written in the HTML file. It is very helpful in presenting text where spaces and line breaks are important like poem or code.

Tip: Text within the Preformated element is typically rendered by the browsers in a monospace or fixed-width font, such as Courier, but you can override this using the CSS font property.
